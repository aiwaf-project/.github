## Hi there 👋

AIWAF is an open security project focused on practical, framework-friendly web application firewall tooling.

### 🙋‍♀️ What we are about

We build WAF components for Python web stacks with optional Rust acceleration:

- `aiwaf` for Python
- `aiwaf-js` for JavaScript
- `aiwaf-php` for PHP
- `aiwaf-rust` for high-performance heuristic helpers

Our goal is to combine rule-based protection with AI-assisted anomaly detection while keeping deployments operable and configurable.

### 🌈 How to contribute

Community contributions are welcome.

- Open an issue for bugs, false positives, or feature proposals
- Submit pull requests with focused changes and tests
- Improve docs and examples for setup, operations, and tuning

When contributing code:

- Keep fallback behavior intact when optional dependencies are unavailable
- Add or update tests for behavior changes
- Document new settings and operational impact

### 🧙 Notes

AIWAF is designed to be modular. You can adopt the Django or Flask package independently, and enable Rust acceleration only when needed.

[DigitalOcean](https://www.digitalocean.com/) provides the cloud infrastructure that powers AIWAF development.
